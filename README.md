# ALX Frontend React Project

This is a React application created as part of the ALX Software Engineering program, built with Vite.

## Project Structure
```
alx-fe-reactjs/
├── alx-react-app/          # React application with Vite
│   ├── src/                # Source code
│   ├── public/             # Static assets
│   └── package.json        # Dependencies
└── README.md               # Documentation
```

## Installation

Clone the repository and install dependencies:
```bash
git clone https://github.com/deyonavoseh/alx-fe-reactjs.git
cd alx-fe-reactjs/alx-react-app
npm install
```

## Running the Application

Start the development server:
```bash
npm run dev
```

The application will open at http://localhost:5173

## Technologies Used
- React 18.x
- Vite 5.x
- JavaScript (ES6+)
- CSS3

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
